# CONFIG DO GITHUB

## Config Inicial:

Inicialização Git:
```
git init
```
Nome para conta:
```
git config --global user.name "NOME DA PESSOA"
```
E-mail do GitHub:
```
git config --global user.email "E-MAIL DO GITHUB"
```
Verificar Status do arquivo:
```
git status
```
Adicionar arquivo:
```
git add "NOME DO ARQUIVO"
```
Criar checkpoint:
```
git commit -m 'primeiro commit'
```
Adicionar Repositório no GitHub:
```
git remote add origin URL_REPOSITORIO_GITHUB
```
Definir a branch main:
```
git branch -M main
```
